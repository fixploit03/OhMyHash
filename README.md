## Instalasi 

```
git clone https://github.com/fixploit03/OhMyHash.git
cd OhMyHash
chmod +x instal.sh
./instal.sh
```

## Cara menggunakan 

1. **Hitung nilai Hash filenya saja**

```
ohmyhash -f [FILE]

atau

ohmyhash --file [FILE]
```

2. **Hitung dan Simpan nilai Hash filenya**

```
ohmyhash -f [FILE] -s [FILE]

atau

ohmyhash --file [FILE] --simpan [FILE]
```
